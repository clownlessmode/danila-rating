# Danila Rating Bot

Telegram-бот для отслеживания социального рейтинга Данилы.

## Команды

- `/danilalox` — минус 10 к рейтингу
- `/danila klass` или `/danilaklass` — плюс 10 к рейтингу  
- `/danilarating` — показать текущий рейтинг и положение

## Положения (примерно каждые 50 баллов)

| Рейтинг | Положение |
|---------|-----------|
| 500+ | Легенда чата |
| 450-499 | Икона стиля |
| 400-449 | Король |
| 350-399 | Босс |
| 300-349 | Мастер |
| 250-299 | Профи |
| 200-249 | Уважаемый |
| 150-199 | Норм пацан |
| 100-149 | Исправляется |
| 50-99 | Пацан |
| 10-49 | Начал путь |
| 0-9 | Нейтрал |
| -10 до -1 | Мусорка |
| -50 до -11 | Позор |
| -100 до -51 | Опущенный |
| -150 до -101 | Дно |
| ... и ниже |

## Запуск

1. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен
2. Активируй venv и установи зависимости:
   ```bash
   source venv/bin/activate
   pip install -r requirements.txt
   ```
3. Запусти: `python bot.py` (токен уже в коде) или `./run.sh`
